 Botpress Chatbot Project 

Hey there! Welcome to my Botpress chatbot repository. I built this conversational agent as a hands-on project to explore how natural language understanding and dynamic dialogue management actually work in practice.

Instead of just a basic Q&A script, this bot is designed to handle multi-turn conversations and figure out user intent without losing track of the context. 

 What it actually does
 *Smart Query Resolution:** Takes user questions and provides immediate, context-aware answers.
*Keeps the conversation going:** Uses structured Botpress workflows so it doesn't get confused if a user asks follow-up questions.
*Handles errors gracefully:** If it doesn't know the answer, it uses fallback routing so the chat doesn't just break.
*Clean Web UI:** Connected to Botpress Webchat v3.6, making it responsive and easy to share.

Tech Stack
* Platform:** [Botpress Cloud](https://botpress.com/)
  * Export Format:** `.bpz` / `.json` configuration
  * Frontend:** Botpress Shareable Webchat
